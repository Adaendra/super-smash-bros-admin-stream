SmashStream

Description

Qu'est-ce que SMASH Stream ?

C'est un outil qui permet d'administrer de manière simple une vue affichant les informations concernant un match. (Nom joueur, score, nom du tour, personnage joué...)

Actuellement, l'outil est composé:

  • D'une page d'admin : Permettant de gérer toutes les informations à envoyer à l'écran de Stream ou de Preview
  • D'une page de stream : Affichant les informations pour le stream. Intégrable facilement sur les logiciels de streaming comme OBS avec son fond vert.
  • D'une page de preview : Similaire à la page de stream, elle permet de vérifier que les informations envoyées sont affichées correctement avant de les envoyer au stream.
  • D'une page de présentation : Pour présenter le prochain match entre les deux joueurs
  • D'un "English-mode" : Pour utiliser l'application en anglais et avec les noms des personnages en anglais

Utilisation

Pour pouvoir utiliser la dernière version du projet, il suffit de télécharger le projet en allant sur cette page, de cliquer sur "Clone or Download", puis sur "Download ZIP". Ou sinon, vous pouvez aller chercher une ancienne version ou une version alternative ici.

Il vous faudra ensuite installer NodeJS.

Puis, via l'interpréteur de commande de Node, se diriger dans le dossier du projet et lancer la commande "node index.js" pour lancer le serveur.

Vous n'avez plus qu'à utiliser les pages "admin", "stream" et "preview" avec votre navigateur.

Un tutoriel vidéo arrivera prochainement.
