Pour pouvoir lancer le serveur, il faut :
- Lancer l'invite de commandes Node.js
- Se déplacer dans le dossier du projet via des commandes suivantes :
	- D: Pour se déplacer sur le disque D
	- cd .. Pour remonter d'un dossier
	- cd projet pour aller dans le dossier projet
- Taper la commande suivante :
    node index.js
- Appuyer sur "Entrée" et le tour est joué !

Pour arrêter le serveur, il suffit de fermer l'invite de commandes.
